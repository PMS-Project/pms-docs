Pms::BaseModule
Pms::Modules::Security::Mod

Description

This is the Baseclass for all moduls/plugins that want to integrate into the PMS admin interface.  It defines a unified API to make the implementation easy.

navbarElements

sub navbarElements

This method should returns a array of hashes containing all navbar elements needed by the module:

sub navbarElements{
  return [{
    NAME => "User bearbeiten",
    HREF => "module.pl?mod=Security;view=addUser"
  },{
    NAME => "Userrechte verwalten",
    HREF => "module.pl?mod=Security;view=userRoles"
  }];
}

sub javascripts

This method should return a array of hashes containing all javascript files required by the module.

sub javascripts{
  my $self = shift or die "Need Ref";
  my $view = shift;
  if($view eq "view1"){
    return [{
      LOCATION => "view1.js"
    }];
  }
}
