Pms::Modules::Security::Mod

Pms::BaseModule
Pms::Modules::Security::Mod

Description

This is the Main Object of the Admin-Interface Security Module.

It has 4 different Views, every View is implemented in its own Backend,

and it uses Hashes to map the backend function names to the action and view parameters that are passed in the URL.
