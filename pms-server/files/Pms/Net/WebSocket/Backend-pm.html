Pms::Net::WebSocket::Backend

To use the code in the Package it just has to be imported.  It will automatically register read and write functions to AnyEvent::Handle so we don’t have to care about how the Handles internally read the Packages from the Socket.  That makes it easy to plug in another Protocol easily.
