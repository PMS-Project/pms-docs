changeNick

sub changeNick

Changes the Nick of a connected User

Note

changeNick will not send a change_nick_request event , this has to be done by the caller

Access

Public

Parameters

$connectionThe User connection Object
$newNickThe New Nickname
$forceIf the nick already exists , force the change (optional, default value is false)

Returns

0for failed
1for success
